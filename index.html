About

RepSnapper is an alternative host software for controlling RepRap (http://reprap.org) open source 3D printers.

Getting it

  • Clone the source to compile on unix/linux/mac: git clone https://github.com/timschmidt/repsnapper.git
  • Update with newer patches later by: git pull
  • source tarball of latest tagged release: https://github.com/timschmidt/repsnapper/tags
  • Windows builds: https://github.com/timschmidt/repsnapper/downloads

Compilation

To build and run repsnapper do:

./autogen.sh

make

./repsnapper

For more details on prerequisites see the manual: https://github.com/timschmidt/repsnapper/blob/master/doc/manual.asciidoc
